# Rails Blog App

This a Microverse project that uses Rails and SQL to create a Blog app with users and posts.


## Built With

- Ruby on Rails
- SQL
## Getting Started

### Installation

1. Clone the repo

   ```sh
   git@github.com:smunozmo/rails-blog-app.git
   ```

### Usage

1. Execute program

   ```sh
   rails s
   ```

### Test

1. For testing the classes run:

   ```sh
   bundle exec rspec  
   ```
